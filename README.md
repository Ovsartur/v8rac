## Установка

Для установки необходимо:
* Скачать файл v8rac.ospx из раздела [releases](https://github.com/khorevaa/v8rac/releases)
* Воспользоваться командой:

```
opm install -f <ПутьКФайлу>
```
или установить с хаба пакетов

```
opm install v8rac
```
